# STM32_PCF8833_LCD

Code to make a STM32F103C drive a PCF8833 based LCD

 * PA0	- RESET (Active Low)
 * PA1	- CE (Active Low)
 * PA8	- USART1 CLK
 * PA9	- USART1 TX
 * PA10	- USART1 RX
